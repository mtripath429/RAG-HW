Sample Document for RAG Testing

This is a test document to ensure the RAG application works correctly.

Key Information:
- This application uses FAISS for semantic search
- It processes PDF and TXT files from the docs folder
- Questions are answered using the OpenAI API with context from your documents
- The app caches the vectorstore for efficiency

How to use:
1. Add PDF or TXT files to the docs/ folder
2. Run the Streamlit app
3. Ask questions about your documents
4. The app will search for relevant passages and generate answers

Example questions:
- What is this document about?
- What information does it contain?
- Can you summarize the key points?
